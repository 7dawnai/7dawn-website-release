# 常见问题

## 为什么设置里没有"浅色主题"选项？

3Studio 目前只有深色界面这一种外观，设置页里没有主题切换开关，也没有计划中的浅色模式。这是当前版本的既定状态，不是缺失或 bug。完整的设置项说明见《[设置与通知](./guides/settings.md)》。

## 为什么 Agent 干活的时候，我没收到系统通知？

只要 3Studio 窗口正处于你当前聚焦查看的状态，系统通知就不会弹出——3Studio 认为你已经在看着界面了，不需要再额外提醒你，这样可以避免正在使用时被打断。把窗口切到后台（比如切换到其他应用）后，Agent 完成任务、出错，或后台任务结束时才会弹出系统通知；点击通知会把 3Studio 窗口带回前台。目前设置里没有单独关闭这些通知的开关。详见《[设置与通知](./guides/settings.md)》。

## 为什么附件传不上去，或者被拒绝了？

单条消息最多能带 **8 个附件**；图片单个不超过 **5MB**，文本类文件（如 `.md` `.txt` `.csv` `.json` `.yaml` `.yml` `.log` `.toml`）单个不超过 **1MB**；其他类型的文件（如 PDF、压缩包、可执行文件）目前不支持作为附件添加。超出数量、超出大小，或文件类型不支持时，界面会弹出提示条说明具体原因，附件不会被添加进去。完整的添加方式（拖拽 / 粘贴 / `@` 引用）见《[与 Agent 对话](./guides/chat.md)》。

## 为什么重启后，之前打开的多个工作区不见了？

左侧工作区切换栏里"当前打开着的多个工作区"这份列表，只在这一次运行期间有效——关闭并重新启动 3Studio 后会清空，回到首页。但你打开过的每个项目都会被记录进首页的"最近"列表，这份列表是长期保留的，不会因为重启消失；从"最近"列表重新点开同一个工作区时，3Studio 还会记住你上次在这个工作区里选中的 Agent 和会话，直接帮你恢复回去。详见《[工作区与文件](./guides/files.md)》。

## 回放里能看到当时的屏幕画面吗？

不能，回放不是屏幕录像。3Studio 记录的是 Agent 当时调用了哪些工具、参数是什么，回看时会按这份记录重新"跑"一遍，在 Computer 面板里重建出当时的工具调用、文件读写等状态，而不是真实屏幕像素的录制。另外截至目前的版本，界面里还没有一个入口可以直接打开某次已完成任务的回放视图——记录本身会在后台自动保存，但暂时无法从界面里回看、播放或跳转到某一步。详见《[回放](./guides/replay.md)》。

## 为什么在 macOS 上找不到 MATLAB / STK 的设置？

MATLAB 与 STK 集成目前只支持 Windows。在 macOS 或 Linux 上，设置页里"工具"标签下不会出现这部分内容，属于当前版本的既定限制，不是配置遗漏。详见《[MATLAB 与 STK 集成](./guides/matlab-stk.md)》。

## 怎么更换 Agent 使用的模型？

模型配置是**全局**的：进入设置页的 **LLM** 标签，里面的 Model Provider 区域配置好之后，所有 Agent 会共用同一份模型配置，无法在某个 Agent 的编辑器里单独指定不同模型。详见《[设置与通知](./guides/settings.md)》。

## 会话记录存在哪里？能导出吗？

会话记录保存在你这台电脑本地，不会自动同步到其他设备或账号。想要导出某次对话，右键点击会话列表里对应的会话，在弹出菜单里选"导出对话"，会打开系统的"保存文件"对话框；导出的是一份 JSON 文件，包含这次对话的基本信息（所属 Agent、工作区、导出时间）和完整的消息记录——其中过长的工具运行输出会被截断，图片等附件只保留文件引用、不含数据本身。详见《[Agent 与会话管理](./guides/agents.md)》。

## 为什么我发了新消息，会话没有排到列表最前面？

会话列表按创建 / 重命名 / 首次打开的时间从新到旧排列，在同一会话里继续发送新消息不会让它重新浮到列表最前面——这是既定的排序规则，不是没有及时刷新。详见《[Agent 与会话管理](./guides/agents.md)》。
