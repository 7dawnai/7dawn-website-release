# 快速上手

本文带你走完第一次使用 3Studio 的完整流程：安装、启动、打开一个项目、认识主界面，再发出第一条消息让 Agent（替你干活的 AI 助手）开始工作。

## 0. 安装 3Studio

> ⚠️ 待核对：安装包获取渠道与系统要求（如支持的操作系统版本、是否需要额外运行环境）。请向管理员或技术支持确认后再进行安装。

## 1. 启动与首页

打开 3Studio 后，你会先看到首页——这里还没有进入任何具体项目，主要用来选择接下来要打开哪个工作区（你选择的本地项目文件夹）：

- 左侧是 3Studio 的品牌标识和版本号。
- 右侧是"工作区"卡片，包含：
  - **打开目录**按钮：用来选择一个本地文件夹作为工作区（也可以用快捷键 `⌘O`／`Ctrl+O`）。
  - **最近**列表：按最后一次打开的时间从新到旧排列，列出你之前打开过的项目，点击某一项即可直接进入。
  - 如果你还没有打开过任何项目，这里会显示一段空状态提示，引导你点击"打开目录"开始。

首页本身不能直接和 Agent 对话——需要先打开一个工作区，进入项目后才能在左侧栏选择 Agent（默认排在第一位的是内置助手小石头 Pebble）开始对话，详见《[Agent 与会话管理](./guides/agents.md)》。

## 2. 打开项目

点击首页的**打开目录**按钮，会弹出系统自带的文件夹选择窗口：

1. 选择一个已有的本地文件夹，或在窗口里新建一个空文件夹。
2. 确认后，3Studio 会把这个文件夹作为当前的**工作区**打开，并自动进入三栏主界面。
3. 这个文件夹路径会被记录到首页的"最近"列表里，之后可以直接从列表点击再次打开，不用每次都重新选择。

3Studio 支持同时打开多个工作区，并通过左侧的工作区切换栏在它们之间来回切换；不过这个"当前打开着的工作区"列表只在本次运行期间有效，关闭并重新启动 3Studio 后会回到首页，需要重新从"最近"列表或"打开目录"进入（"最近"列表本身是长期保留的，不会因为重启而消失）。工作区与文件的完整说明见《[工作区与文件](./guides/files.md)》。

## 3. 认识三栏界面

进入工作区后，主界面分为三栏：

- **左侧栏（Sidebar）**：包含 Agent 与会话列表（在这里创建新任务、切换 Agent、查看历史会话）、当前工作区的文件树，以及底部的运行时状态入口——点击"运行时"可以展开一个面板，查看本地运行时、MCP 工具连接、以及 MATLAB / STK 等模块的连接状态。
- **中间对话区**：和当前选中 Agent 的对话界面，输入框在底部。
- **右侧 Computer 面板**：展示 Agent 执行操作的过程，包含 **Tools（工具调用）**、**Files（文件）**、**Terminal（终端）** 三个标签页。

三栏的宽度都可以调整：

- 把鼠标移到两栏之间的分隔线上，按住并拖动即可改变左右宽度。
- 双击分隔线可以把宽度恢复到默认比例。
- 左侧栏和右侧 Computer 面板都可以收起：把分隔线拖到最边上即可收起；收起后，屏幕边缘会出现一个小箭头按钮，点击可以重新展开。
- 你调整过的宽度会被记住，下次打开 3Studio 时还是你上次调整后的样子。

## 4. 第一次对话

一切就绪后，就可以开始你的第一次对话了：

1. 在左侧栏确认当前选中了想要使用的 Agent（默认已经帮你选好一个）。
2. 在中间对话区底部的输入框里，描述你想让 Agent 做的第一个任务，然后发送。
3. Agent 会以流式方式逐字返回它的思考和回复。
4. 当 Agent 执行工具调用（比如读写文件、跑命令）时，右侧 Computer 面板会自动切换展示对应的操作过程，你可以随时点开 Tools / Files / Terminal 标签查看细节。

关于对话输入框的具体用法（如引用文件、斜杠命令等），见《[与 Agent 对话](./guides/chat.md)》；关于 Computer 面板三个标签页的详细说明，见《[Computer 面板](./guides/computer-panel.md)》。
