# 回放

3Studio 会自动记录 Agent 在每个任务里做过的每一步操作，方便事后回看它到底做了什么。本页说明这项能力目前是怎么工作的。

## 回放是什么，不是什么

- **是**：对 Agent 操作序列的忠实记录——它调用了哪些工具、什么时候调用的、参数是什么。回看时，3Studio 会把这些操作按记录的顺序重新"跑"一遍，在 [Computer 面板](./computer-panel.md) 里重建出当时的工具调用、文件读写等状态。
- **不是**：屏幕录像或截图。回放看到的是**根据记录重新计算出来的面板状态**，不是当时屏幕画面的原始像素；如果某个文件内容后来又被改动过，回放时看到的仍然是记录里保存的那个版本，跟当前工作区文件的实际内容无关。

## 记录是怎么发生的

任务在 Agent 界面里跑起来后，每一步操作都会被自动记录，不需要你手动开启。当这一轮任务结束（Agent 完成任务）时，这次任务的完整操作记录会被保存到你这台电脑本机的浏览器数据库里，具体见下方"注意事项"。

截至目前的版本，界面里还没有一个按钮或入口可以让你打开某个已完成任务的回放视图——记录本身会在后台自动保存，但暂时无法从界面里直接回看、播放或跳转到某一步。

> ⚠️ 待核对 / 当前状态：这部分交互（时间线播放 / 暂停 / 逐步跳转）后续是否以及何时开放尚未确定，需团队评估；如果决定开放，本页会更新为完整的操作说明。

## 注意事项

- 回放记录保存在**本机**（你当前使用的这台电脑）：不会同步到其他设备，换一台电脑打开同一个工作区看不到之前留下的记录。
- 记录只保存在本机的浏览器数据库里，不会写入你工作区里的项目文件。
