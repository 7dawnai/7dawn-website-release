# Agent 与会话管理

Agent（替你干活的 AI 助手）是 3Studio 里实际帮你完成工作的角色。每个工作区（你打开的项目文件夹）都可以拥有多个 Agent，每个 Agent 又可以维护多条独立的会话（一段对话记录）。本页说明如何创建、配置 Agent，认识内置助理 Pebble，以及如何管理会话。

## 什么是 Agent

打开一个工作区后，左侧 Sidebar 顶部会有一个 "Agents" 区域，列出当前可用的 Agent。每个 Agent：

- 有自己的名称和一份"工作说明书"（决定它的角色定位与行为方式，详见下一节）
- 拥有一份独立的会话列表——你和它的每一次对话都会被单独保存
- 可以随时点击切换、编辑或删除（内置助理 Pebble 除外，见下文）

## 创建与编辑 Agent

### 创建 Agent

点击 Sidebar 里 Agent 列表下方的"新建 Agent"，会打开一个模板选择弹窗。**新建 Agent 不是从空白开始手填名称**，而是从内置模板中挑选一个作为起点——常见模板包括"卫星总师""KiCad 设计师""编程助手"等（实际可选模板取决于你安装的版本）。

弹窗会展示所选模板的说明（角色描述、随附的技能）供预览，名称字段在这一步是锁定的（显示为只读，防止误改导致模板配置错乱）。确认"创建"后，3Studio 会立即为这个新 Agent 打开编辑器，方便你在此基础上进一步调整。

### 编辑 Agent

点击 Agent 行右侧的编辑图标（鼠标悬停在该行上才会出现），进入编辑器，可以修改：

- **名称**：Agent 在 Sidebar 里显示的名字
- **说明文档**：一份 Markdown 文档，定义这个 Agent"是什么角色、该怎么做事"。对你自己创建的 Agent，这份文档就是它的行为指令（AGENTS.md）；对内置助理 Pebble，这份文档叫 "Soul Prompt"（人设提示词），作用相同。文档区提供"编辑 / 预览"两种模式切换，方便你确认排版效果

> ⚠️ **没有"选择模型"字段**：Agent 编辑器里不能给单个 Agent 单独指定模型。3Studio 的模型是全局设置，所有 Agent 共用同一个模型配置，修改请去[设置页](./settings.md)。

已创建的 Agent（非内置助理 Pebble）可以点击"删除"移除；删除前会弹窗二次确认。

## 内置助理 Pebble（小石头）

Pebble（小石头）是 3Studio 自带的默认助理，具有以下特点：

- **跨项目共享**：不管你打开哪个工作区，Pebble 都在，它的会话不属于某一个特定项目
- **Sidebar 置顶**：在任意工作区的 Agent 列表里，Pebble 始终排在最前面
- **可以编辑，不能删除**：你可以修改它的名称和 Soul Prompt（人设提示词），但删除按钮不会出现在它的编辑器里

## 会话管理

展开某个 Agent（点击它的名称即可切换到它并展开会话列表；点击行首的箭头图标可单独收起/展开，不切换 Agent），下方就是它的会话列表。

- **新建会话**：鼠标悬停在 Agent 行上，会出现一个"+"图标，点击即可新建一条会话并立即切换过去
- **切换会话**：直接点击列表里的某条会话
- **重命名**：双击会话名称，或悬停后点击铅笔图标，或右键会话弹出菜单选"重命名"（快捷键 F2）；输入新名称后按 Enter 确认，按 Esc 放弃修改，点击别处（失焦）则保存当前输入
- **删除**：右键会话，在弹出菜单里选"删除"，会弹窗二次确认
- **导出对话**：右键会话，在弹出菜单里选"导出对话"，会打开系统的"保存文件"对话框，让你选择保存位置。导出的是一份 JSON 文件（文件名形如 `<Agent 名>-<会话标题>-<日期>-<时间>.json`），内容包含这次对话的基本信息（所属 Agent、工作区、导出时间）和完整的消息记录；其中过长的工具调用输出会被截断，附件图片只保留引用、不含图片数据本身
- **排序**：会话列表按创建 / 重命名 / 首次打开时间从新到旧排列；在同一会话里继续发送新消息不会让它重新浮到列表最前面
- **展开更多**：每个 Agent 默认最多显示最近 5 条会话，超出的部分会收进一行"更多（N）"提示；把鼠标悬停在这行上，会弹出浮层显示其余的会话

会话标题会根据你发送的第一条消息内容自动生成；发送第一条消息之前，会话名称会显示为 `Untitled`。
