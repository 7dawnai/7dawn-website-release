# 与 Agent 对话

对话是你和 **Agent**（替你干活的 AI 助手）打交道的主要方式：无论是在某个工作区里干活的 Agent，还是内置的私人助理 Pebble，界面下方都是同一套输入框和消息列表。本页说明输入框的各项能力，以及消息区里会出现的各种交互卡片。

## 发送消息与查看回复

在窗口下方的输入框里打字，按 **Enter** 发送；按 **Shift + Enter** 只换行、不发送。

发送后，Agent 的回复会以**流式**方式逐字显示，不用等它想完整句话再一起出现。回复生成过程中你可能会看到：

- **思考中**：一个带旋转图标的小卡片，标注"思考中"和已经过去的秒数。思考结束后会变成"已思考 N 秒"；如果这次思考产生了可读内容，卡片会变成可点击的，点一下展开或收起具体的思考文字。
- **工具调用**：Agent 读写文件、执行命令等操作会以小方块（chip）的形式显示。连续调用同一种工具（比如连续创建了 3 个文件）会自动折叠成一组，标题类似"创建了 3 个文件"，点击标题展开查看每一次具体调用；点击某一次具体调用可以跳转到右侧 **Computer 面板**（展示 Agent 操作过程的面板，详见 [Computer 面板](./computer-panel.md)）的 Tools 标签页查看详情。

Agent 处理中时输入框仍然可以继续打字，但按 Enter 不会发出去（安静地不做任何事，也不会提示错误）；此时输入框右下角原本的发送按钮也会变成**停止**按钮，不再有发送按钮可点。想要打断当前任务见下文"中止正在执行的任务"。

## 添加附件

单条消息最多可以带 **8 个附件**：图片不超过 **5MB**，文本类文件（如 `.md` `.txt` `.csv` `.json` `.yaml` `.yml` `.log` `.toml`）不超过 **1MB**。超出数量或大小限制、或文件类型不支持时，界面右上角会弹出提示条说明原因，附件不会被添加。

三种添加方式：

1. **点击输入框左下角的回形针图标**，从系统文件选择器里多选文件。
2. **把文件拖进输入框区域**，松开鼠标即可（拖入时会看到"松开以添加附件"的提示）。
3. **直接粘贴剪贴板里的图片**（例如截图后按 Ctrl/Cmd + V），仅支持图片。

添加成功的附件会显示为一个个"药丸"标签，标注文件名和大小，点标签上的 × 可以移除；正在读取的附件会显示加载动画，读取完成前无法发送消息。

## 用 `@` 引用文件或目录

在输入框里输入 `@`，接着打几个字符，就会弹出当前工作区内匹配的文件 / 目录列表（模糊匹配文件名，最多显示 8 条，隐藏文件如 `.git` 不会出现）。用上下箭头选择，按 Enter 或 Tab 确认插入引用。

如果你 `@` 引用的是一张图片，它会被自动当作**附件**发送（而不是一段文字引用），这样可以直接把截图丢给 Agent 让它"看"。

> 引用文件需要当前已经打开一个工作区（项目）。

## 斜杠命令

在输入框里输入 `/` 会弹出命令列表，用上下箭头选择、Enter 或 Tab 确认，支持按名称前缀过滤。列表最多分四组：

- **SYSTEM**（固定内置，始终可用）
  - `/compact` —— 压缩当前对话的上下文，长对话快占满时用它腾出空间。
  - `/memory` —— 查看当前 Agent 保存下来的记忆内容。
  - `/mcp` —— 查看已连接的 MCP 服务器、工具和提示词状态清单。
- **COMMANDS / SKILLS / MCP**（随当前 Agent 而变化）—— 来自这个 Agent 自身配置的自定义命令、可调用的技能（Skill），以及已连接 MCP 服务提供的提示词。具体能看到哪些命令取决于当前选中的 Agent 和它的权限设置：某些技能可能被这个 Agent 的权限白名单限制而不出现在列表里，不同 Agent、不同工作区看到的命令清单可能不一样。

斜杠命令只要选中了 Agent 并且已经有一个对话（会话）存在就能用。

## 中止正在执行的任务

Agent 处理中（思考、生成回复、调用工具、压缩上下文等）时，输入框右下角的发送按钮会变成一个**停止**按钮。点它会立即中止当前这一轮任务：已经生成出来的部分内容会保留在对话记录里，不会被撤销。中止是立刻生效的，不需要等 Agent 自己收尾。

## 批准或拒绝 Agent 的操作请求

有些操作（比如执行命令）需要你确认后 Agent 才会继续执行。此时对话里会出现一张**审批卡片**，标题是要执行的操作摘要，下面是具体参数（内容较长时可以点击"展开 / 收起"查看全文）。卡片提供三个按钮：

- **允许**：批准这一次操作。
- **拒绝**：驳回这一次操作，Agent 会知道你不同意。
- **始终允许**：批准这一次，并记住这类操作，之后 Agent 遇到同类请求时不再重复弹卡片询问你。

另外还有一种**提问卡片**：当 Agent 需要你补充信息才能继续时会弹出来，每个问题给出若干选项（可能允许单选或多选），部分问题下方还有一个文本框可以手动输入自定义答案。回答完所有问题后点击"提交"继续，或者点击"跳过"让 Agent 自己决定。
