# 工作区与文件

工作区就是你选择打开的本地项目文件夹。本页说明工作区的含义、如何打开和切换工作区（包括同时打开多个），以及如何用左侧的文件树管理文件。

## 工作区是什么

打开 3Studio 后，你首先要做的事就是选择一个本地文件夹作为工作区（参见《[快速上手](../getting-started.md)》第 2 步）。选定之后：

- 左侧栏会展示这个文件夹里的文件和子文件夹（文件树）。
- 当前工作区里的 Agent（替你干活的 AI 助手）能读写的文件范围，限定在这个文件夹以内——不会跑到你电脑上的其他位置读写文件。
- 每个工作区都有自己独立的 Agent 列表和会话记录，详见《[Agent 与会话管理](./agents.md)》。

## 打开与切换工作区

### 打开工作区

在首页点击"打开目录"按钮（或直接从"最近"列表点选之前打开过的项目），即可把一个本地文件夹作为工作区打开，详细步骤见《[快速上手](../getting-started.md)》。

### 同时打开多个工作区

3Studio 支持同时打开多个工作区，并在它们之间来回切换，不需要先关掉当前项目：

1. 点击左侧栏底部（用户头像左侧）的一个小图标按钮，展开"工作区切换栏"——这是一条固定在最左侧的窄栏，列出本次运行期间你打开过的所有工作区。
2. 切换栏底部有一个"添加工作区"按钮：点击后可以从"最近项目"里搜索并选择一个，或点击"添加本地文件夹…"重新打开系统的文件夹选择窗口，加入一个新的工作区。
3. 在切换栏里点击某个工作区名称即可切换过去，当前所在的工作区会高亮显示。
4. 把鼠标悬停在某个工作区行上，会出现一个"···"按钮，打开菜单可以"切换到此工作区"或"从列表中移除"——移除只是把它从这条切换栏里拿掉，**不会删除或修改你本地的项目文件**。

> ⚠️ **这份"已打开工作区"列表不会保留到下次启动**：关闭并重新打开 3Studio 后，会回到首页，需要重新从"最近"列表或"打开目录"进入。但"最近"列表本身是长期保留的，不会因为重启而消失，且重新打开同一个工作区时，3Studio 会记住你上次在这个工作区里选中的 Agent 和会话。

## 文件树操作

左侧栏的"文件"区域展示当前工作区的文件树；点击区域标题可以整体收起/展开这个区域。文件夹前的箭头图标表示可以点击展开/收起，文件则可以直接单击选中并在右侧 Computer 面板的 Files 标签页里查看内容。

右键点击文件树里的项目会弹出菜单，菜单内容根据你右键的目标略有不同：

**右键点击空白区域**（会在工作区根目录下创建）：
- 新建文件
- 新建文件夹

**右键点击某个文件夹**：
- 新建文件 / 新建文件夹（会在这个文件夹内部创建）
- 重命名（菜单标注快捷键 F2）
- 删除（危险操作，见下方说明）
- 复制路径

**右键点击某个文件**：
- 打开（菜单标注快捷键 Enter；也可以直接单击文件名）
- 重命名（菜单标注快捷键 F2）
- 删除（菜单标注快捷键 Del，危险操作，见下方说明）
- 复制路径
- 复制相对路径（相对于工作区根目录的路径，比如 `src/main.py`，而不是完整的本地路径）
- 在 Finder 中显示（会打开系统自带的文件管理器并定位到这个文件；即使在 Windows 系统上，这一项目前也显示"Finder"字样）

> ⚠️ 待核对：右键菜单里标注的 F2 / Del / Enter 快捷键提示，目前只在菜单已经打开、你把鼠标移到对应菜单项时才能点击生效；未确认是否支持在不打开菜单、仅选中文件/文件夹的情况下直接按这些键触发同样的操作。建议按提示先右键调出菜单再选择相应项。

**新建文件 / 新建文件夹**：创建后会立即进入重命名状态，光标已经选中默认名称（新文件默认叫 `untitled`，新文件夹默认叫 `untitled-folder`），直接输入你想要的名字，按 Enter 确认，按 Esc 取消重命名（但文件/文件夹本身仍会保留刚才的默认名）。

**删除**：无论删除文件还是文件夹，都会先弹出一个确认弹窗（标题为"删除文件？"或"删除文件夹？"，并提示"《文件/文件夹名》将被永久删除"），点击"删除"按钮才会真正执行，避免误删。

工作区之外的改动（比如你在系统的文件管理器里直接增删改了工作区文件夹里的文件）也会被自动识别：文件树会实时刷新，显示最新的文件和文件夹结构，不需要手动刷新页面。

## Agent 对文件的改动如何呈现

Agent 执行任务时可能会直接读写工作区里的文件（比如创建新文件、修改代码、生成文档）。这些改动会体现在两个地方：

- **左侧文件树**：新增或修改的文件会自动出现在文件树里，和前面提到的"外部改动自动刷新"是同一套机制。
- **右侧 Computer 面板的 Files 标签页**：Agent 操作过或你手动点开的文件内容会展示在这里，方便你确认改动内容。完整说明见《[Computer 面板](./computer-panel.md)》。
