# Computer 面板

Computer 面板是窗口右侧的一块区域，用来实时呈现 Agent 正在做的具体事情——它执行了什么命令、读写了哪个文件、终端里发生了什么。和左边的对话区不同，对话区只显示 Agent 的文字回复和一个简要的"工具调用"小方块，Computer 面板展示的是这些操作背后的完整细节。

面板顶部有三个标签页：**工具**（Tools）、**文件**（Files）、**终端**（Terminal）。Agent 干活时它通常会自动帮你切到该看的那个标签页，但你也可以随时自己点开某一个查看。

## 工具标签页 — 看 Agent 正在执行的命令与输出

工具标签页按时间顺序列出 Agent 做过的每一次工具调用（执行命令、读文件、写文件、改文件等），上方是一份可滚动的概览列表，下方是当前选中这一条的详细内容。

概览列表上方有几个筛选和跳转工具：

- 按状态筛选：**全部 / running（运行中）/ error（出错）/ done（完成）**
- 按名字搜索：输入框可以按工具名称过滤列表
- 快捷跳转：**第一个错误**、**运行中**、**最新** 三个按钮，点击直接跳到对应位置

点击列表里的某一条，下方会显示它的详情：

- 执行命令类工具（比如运行一条 shell 命令）：显示具体的命令文本，以及执行过程中不断增长的输出内容——命令还没跑完时你能看到输出**实时滚动**，不用等它跑完
- 读写文件类工具：显示涉及的文件路径和具体内容
- 其他类型的工具调用：显示传给它的参数和返回的结果（原始数据形式）

每条记录右上角有一个状态标记，对应 running / done / error 三种状态之一。

列表和详情之间有一条可以上下拖动的分隔线，拖动它可以调整概览列表的显示高度，这个高度会被记住，下次打开还是你调整过的样子。

底部还有上一步 / 下一步 / 跳转到最新 三个导航按钮，效果和点击概览列表里的某一行、或点顶部的快捷跳转按钮是一样的，只是提供了另一种翻看方式。

## 文件标签页 — 浏览和编辑文件

文件标签页展示某一个具体文件的内容，通常通过左侧文件树点开一个文件后自动切换过来（文件树的完整用法见 [工作区与文件](./files.md)）；点击对话里 Agent 引用过的文件也会跳到这里打开。

打开的文件长什么样，取决于它的类型：

- **Markdown 文件**（`.md` `.mdx` `.markdown` `.txt`）：默认以排版后的预览模式展示，右上角有一个"预览 / 编辑"切换，切到编辑模式后可以直接改动内容，按 **Ctrl/Cmd + S** 保存
- **图片文件**：以只读方式居中显示，底部显示文件名、像素尺寸和文件大小；单个文件超过 20MB 无法预览，会提示文件过大，并给出一个按钮在系统文件管理器里定位这个文件
- **SVG 矢量图文件**：默认按图片方式预览（Preview），也可以切到源码模式（Source）用代码编辑器查看和修改 SVG 的 XML 源码，同样支持 Ctrl/Cmd + S 保存
- **其他代码 / 文本文件**：用内置代码编辑器打开，按文件扩展名自动做语法高亮（`.m` 文件会按 MATLAB 语法高亮），同样支持 Ctrl/Cmd + S 保存修改

> 图片和 SVG 的预览就集成在这个文件标签页里，并不是一个单独的预览标签页。

还没有打开任何文件时，这里会提示你去文件树选一个文件，或者点击工具标签页里涉及的文件路径。

## 终端标签页 — 内嵌终端

终端标签页是一个内嵌在应用里的命令行终端，你可以在这里手动敲命令，效果和在系统自带的终端程序里操作一样。打开一个工作区后，只要还没有运行中的终端，3Studio 会自动为你新建一个。

顶部是终端会话的标签条，可以同时开多个终端会话，逐一切换或关闭：

- 点 **+** 新建一个终端会话
- 点会话标签上的 **×** 关闭它
- **清除**（橡皮擦图标）清空当前终端的显示内容
- **复制全部输出**（复制图标）把当前终端里的全部内容复制到剪贴板

**向 Agent 询问终端输出**：用鼠标在终端里选中一段文字，再点工具栏上的闪光图标（Sparkles），3Studio 会把这段输出连同上下文自动整理成一条消息发给当前 Agent，请它帮你解释这段内容——不用自己手动复制粘贴。

**Agent 也可以帮你把命令准备好**：如果 Agent 的回复里包含一段命令代码块，代码块上方会出现一个"插入到终端"按钮，点击后会把这条命令**插入**到终端里（不会自动执行），你确认无误后自己按 Enter 才会真正运行。如果这条命令被判断有一定风险（比如涉及删除、联网等操作），插入前还会先弹出一个确认框，你确认后才会插入。

**移动终端的位置**：终端默认显示在 Computer 面板里，你也可以点击"移至底部停靠栏"按钮把它挪到窗口底部、横跨整个应用宽度显示，这样能获得更宽敞的操作空间；再点"移至面板"可以移回 Computer 面板。移到底部停靠栏后还多两个按钮：**搜索终端输出**和**重命名终端**（给这个终端会话起个好记的名字）。不管在哪个位置，都可以点**最大化**按钮把终端放大到接近全屏，再点一次还原。

## 自动切换与手动锁定

Computer 面板默认会跟着 Agent 的动作自动切换标签页：Agent 一开始执行工具调用，面板就自动跳到工具标签页；这样你不用一直盯着它，Agent 做什么就看什么。

但只要你**自己手动**点了某个标签页，或者点开了某个文件，面板就会记住"你现在在看这里"，暂时**不再**因为新的工具调用自动跳走——这样你翻看历史记录或者仔细看一份文件的时候，不会被打断。

这个手动锁定会在以下几种情况下自动解除，重新回到自动跟随模式：

- 你给 Agent **发送了一条新消息**
- 你**切换到了另一个会话**
- 你**切换到了另一个 Agent**

锁定状态是按会话记的，只影响你正在锁定的那个会话；切到别的会话看到的仍然是自动跟随。

## 折叠面板

如果暂时不需要看 Computer 面板，可以把它收起来腾出空间给对话区：把鼠标移到对话区和 Computer 面板之间的分隔线上，按住往右拖到底，面板就会收起。收起后，窗口右侧边缘会出现一个小箭头按钮，点一下就能把面板重新展开。
